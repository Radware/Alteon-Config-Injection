# Alteon-Config-Injection
The following Ansible playbook allows you to deploy Alteon VM on VMware vCenter and inject configuration to your new Alteon VM.

# Requirements
-	Config file must be named "iso_cfg.txt"
- Ansible >= 2.7
- Python >= 2.6
